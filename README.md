# Pokémon Explorer ⚡

## 📌 Project Overview

Pokémon Explorer is a web application that allows users to search and explore Pokémon using real-time data from the public **PokéAPI**. The application fetches Pokémon information dynamically and displays details such as name, image, and types in a user-friendly interface.

The goal of this project is to demonstrate understanding of **JavaScript, API integration, array higher-order functions, and UI development using HTML, CSS, and JavaScript**.

This project is being developed as part of the course milestone project and will evolve through multiple stages including API integration, interactivity features, and deployment.

---

## 🎯 Purpose of the Project

The purpose of this project is to:

* Practice integrating **public APIs using fetch()**
* Use **JavaScript Higher Order Functions** like `map()`, `filter()`, and `sort()`
* Build a **responsive user interface**
* Create an interactive web application where users can explore Pokémon data dynamically

---

## 🌐 Public API Used

This project uses the **PokéAPI**, a free and open API providing Pokémon data.

API Website:
https://pokeapi.co/

Example Endpoint:

```
https://pokeapi.co/api/v2/pokemon/ditto
```

Example Data Retrieved:

* Pokémon name
* Pokémon image (sprite)
* Pokémon types
* Pokémon stats

## ✨ Planned Features

### 🔎 Search

Users will be able to search for a Pokémon by entering its name.

### 🎯 Filtering

Users will be able to filter Pokémon based on their type (e.g., Fire, Water, Grass).

### 🔼 Sorting

Pokémon can be sorted alphabetically or by ID.

### 🎮 Button Interactions

The application will include interactive elements such as:

* View Pokémon details
* Favorite Pokémon

### 🌙 Dark Mode

Users will be able to toggle between light mode and dark mode for better usability.
